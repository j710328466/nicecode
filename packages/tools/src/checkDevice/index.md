## 环境判断：checkDevice

<Alert type="info">
  <strong>判断环境</strong>
</Alert>

### isIos

```js
import { checkDevice } from '@nicecode/func';

const Demo = () => {
  const isIos = checkDevice.isIos();

  return <div>是否在 Ios 设备端：{isIos}</div>;
};

export default Demo;
```

### isAndroid

import { checkDevice } from '@nicecode/func';
