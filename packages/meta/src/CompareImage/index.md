## API

| 参数 | 说明 | 类型 | 默认值 | 版本 |
| --- | --- | --- | --- | --- |
| label | 左上角标题 | string（必填） | - |  |
| openRoll | 是否支持鼠标滚动放大缩小 | boolean | - |  |
| urls | 链接数组 | string[] | - |  |
| width | 宽度 | string（选填） | - |  |
| height | 高度 | string（选填） | - |  |
| score | 相似度 | string（选填） | - |  |
